# QuizReviewer
QuizReviewerは、過去に解いたクイズを登録することで、"早押しクイズ形式での復習"ができるWebアプリです。<br>
これにより本アプリでは、早押しクイズアプリ等で出た問題の復習を実践的かつ効率的に行うことができます。<br>
制作者は同様の形式で勉強したことで、「みんなで早押しクイズ」というクイズアプリでSランクになることができました（上位１％）。
